# ConectED

ConectED is a full-stack Question and Answer platform designed to connect curious minds with knowledgeable experts. This repository contains both the frontend (Angular) and backend (Node.js with Express) components of the application.

## Technologies Used

### Frontend
- Angular 20.x
- TypeScript
- HTML5, CSS3
- Socket.IO Client

### Backend
- Node.js
- Express.js
- MongoDB (via Mongoose)
- JWT for authentication
- Socket.IO for real-time communication
- Swagger for API documentation

## Project Structure

- `backend/`: Contains the Node.js Express API.
- `frontend/`: Contains the Angular client application.

## Getting Started

Follow these instructions to set up and run the project locally.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or Yarn
- MongoDB (local or cloud instance)

### 1. Clone the Repository

```bash
git clone https://github.com/subrahmanya01/conect_ed.git
cd conect_ed
```

### 2. Backend Setup

Navigate to the `backend` directory, install dependencies, and start the server.

```bash
cd backend
npm install
# Or yarn install
```
